Koppel je Supabase database. Dit is éénmalig — gegevens worden lokaal bewaard.

Stap 01 — Tabellen aanmaken
Open SQL Editor in Supabase, klik "SQL Script bekijken" hieronder, plak en voer uit.
Stap 02 — API sleutels invullen
Ga naar Settings → API en kopieer Project URL en anon public.
